XDT.php : 748

Build a list of DOM elements maintaining the parental relationship.

And defines appropriate methods to work on the list elements.

This class contains jQuery like methods for document traversing and document altering. It implements IteratorAggregate and ArrayAccess interfaces, which allow class objects to behave like an array.

Changes made by a method of this class are not automaticaly saved. Explicitly call the XDT close method to save change.
